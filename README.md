# Task Manager Frontend

Simple frontend application for the **Task Manager** system.

Users can:

* Register account
* Login
* Create tasks
* Update tasks
* Delete tasks
* Mark tasks as completed

This frontend connects to the backend API.

---

## Tech Stack

* React
* TypeScript
* Vite
* TailwindCSS
* Axios
* React Router

---

## Installation

Clone the repository

git clone https://github.com/MHASBIANUR/task-manager-frontend.git

Go to project folder

cd task-manager-frontend

Install dependencies

npm install

---

## Environment Variables

Create `.env` file

Example:

VITE_API_URL=http://localhost:3000

For production:

VITE_API_URL=https://task-manager-backend-production-7d8c.up.railway.app

---

## Run Development

npm run dev

Open in browser:

http://localhost:5173
